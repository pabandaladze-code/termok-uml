# Class Diagram: Система «Теремок» (Управление доступом)

## Обзор

Эта диаграмма классов описывает объектно-ориентированную структуру системы управления доступом в теремок. Основные сущности: сам Теремок (хранит состояние и лимиты) и абстрактный класс Зверь, от которого наследуются все персонажи сказки.

---

## Классы

### Теремок

| Атрибут | Тип | Описание |
|---------|-----|----------|
| maxWeight | int | Максимальный суммарный вес жильцов (100) |
| currentOccupants | int | Текущее количество жильцов |
| totalWeight | int | Текущий суммарный вес всех жильцов |

| Метод | Возвращает | Описание |
|-------|------------|----------|
| добавитьЗверя(зверь: Зверь) | bool | Добавляет зверя, если есть место. Для Медведя → false |
| разрушить() | void | Разрушает теремок, все жильцы покидают |

---

### Иерархия зверей

| Класс | Тип | Вес | Особенности |
|-------|-----|-----|-------------|
| Зверь | abstract | - | Абстрактный класс с методом запроситьВход() |
| Мышка | concrete | 1 | Самый лёгкий |
| Лягушка | concrete | 5 | |
| Зайчик | concrete | 10 | |
| Лисичка | concrete | 15 | |
| Волчок | concrete | 30 | |
| Медведь | concrete | 9999 | Игнорирует проверку веса, разрушает теремок |

---

## Связи между классами

| Связь | Тип | Описание |
|-------|-----|----------|
| Теремок → Зверь | Агрегация (1 → *) | Один теремок содержит много зверей |
| Зверь ← Мышка и др. | Наследование | Все конкретные звери наследуют абстрактный класс Зверь |

---

## Диаграмма классов

```plantuml
@startuml
!theme blueprint
skinparam classAttributeIconSize 0

' --- Основные сущности ---
class Теремок {
    - maxWeight: int = 100
    - currentOccupants: int = 0
    - totalWeight: int = 0
    --
    + добавитьЗверя(зверь: Зверь): bool
    - проверитьВместимость(зверь: Зверь): bool
    + разрушить(): void
}

' --- Иерархия зверей ---
abstract class Зверь {
    - имя: String
    - вес: int
    --
    + запроситьВход(теремок: Теремок): void
}

class Мышка extends Зверь {
    + вес = 1
}

class Лягушка extends Зверь {
    + вес = 5
}

class Зайчик extends Зверь {
    + вес = 10
}

class Лисичка extends Зверь {
    + вес = 15
}

class Волчок extends Зверь {
    + вес = 30
}

class Медведь extends Зверь {
    + вес = 9999
    --
    + запроситьВход(теремок: Теремок): void
}

' --- Связи ---
Теремок "1" o-- "*" Зверь : содержит
Зверь <|-- Мышка
Зверь <|-- Лягушка
Зверь <|-- Зайчик
Зверь <|-- Лисичка
Зверь <|-- Волчок
Зверь <|-- Медведь

note right of Теремок::добавитьЗверя
  **Управление доступом:**
  Если зверь = Медведь → разрушить()
  Иначе проверить totalWeight + вес <= maxWeight
  При успехе увеличить currentOccupants и totalWeight
end note

note bottom of Медведь::запроситьВход
  Медведь игнорирует проверку веса
  и сразу вызывает теремок.разрушить()
end note
@enduml
