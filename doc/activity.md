# Activity Diagram: Система «Теремок» (Управление доступом)

## Обзор

Эта диаграмма активности описывает алгоритм обработки запроса зверя на поселение теремка. Основные ветвления: является ли зверь Медведем, и не превышает ли общий вес лимит.

---

## Алгоритм

| Шаг | Действие | Результат |
|-----|----------|-----------|
| 1 | Зверь делает запрос на поселение | Запуск процесса |
| 2 | Проверка: это Медведь? | Если да → переход к разрушению |
| 3 | Проверка: totalWeight + вес_зверя ≤ maxWeight (100) | Если да → заселение, если нет → отказ |
| 4 | Заселение | Обновление currentOccupants и totalWeight |
| 5 | Разрушение | Теремок уничтожен, процесс остановлен |
