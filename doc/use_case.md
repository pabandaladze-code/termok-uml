# Use Case Diagram: Система «Теремок» (Управление доступом)

## Обзор

Эта диаграмма вариантов использования описывает взаимодействие акторов (Мышка, Лягушка, Заяц, Лиса, Волк, Медведь) с системой «Теремок» для управления доступом и контроля вместимости.

---

## Акторы

| Актор | Описание |
|-------|----------|
| **Мышка** | Маленький зверёк (вес = 1). Запрашивает поселение. |
| **Лягушка** | Маленький зверёк (вес = 5). Запрашивает поселение. |
| **Заяц** | Средний зверёк (вес = 10). Запрашивает поселение. |
| **Лиса** | Средний зверёк (вес = 15). Запрашивает поселение. |
| **Волк** | Крупный зверёк (вес = 30). Запрашивает поселение. |
| **Медведь** | Огромный зверь (вес = 9999). Игнорирует проверку веса и разрушает теремок. |

---

## Варианты использования

| Вариант использования | Описание |
|-----------------------|----------|
| **Запрос на поселение** | Зверь обращается к теремку с просьбой пустить его внутрь. |
| **Проверка свободных мест** | Система проверяет, не превысит ли общий вес лимит (максимум = 100). |
| **Разрушить теремок** | Медведь игнорирует проверку веса и разрушает теремок. |

---

## Связи

| Связь | Тип | Описание |
|-------|-----|----------|
| Запрос на поселение → Проверка свободных мест | `<<include>>` | При каждом запросе выполняется проверка свободных мест. |
| Запрос на поселение → Разрушить теремок | `<<extend>>` | Только если зверь = Медведь. |

## Диаграмма вариантов использования

```plantuml
@startuml Теремок_UseCase
!theme blueprint

actor Лиса as Fox

rectangle "Управление доступом" {
  Mouse --> (Запрос на поселение)
  Frog --> (Запрос на поселение)
  Hare --> (Запрос на поселение)
  Fox --> (Запрос на поселение)
  Wolf --> (Запрос на поселение)
  Bear --> (Запрос на поселение)

  (Запрос на поселение) ..> (Проверка свободных мест) : <<include>>
  (Запрос на поселение) ..> (Разрушить теремок) : <<extend>>
}

note right of (Разрушить теремок)
  Медведь игнорирует
  проверку веса и разрушает
  теремок.
end note

note left of (Проверка свободных мест)
  Проверка:
  totalWeight + вес_зверя <= 100
  Если да → заселение,
  если нет → отказ.
end note
@enduml
